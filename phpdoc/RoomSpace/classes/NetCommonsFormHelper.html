input()

input(string  $fieldName, array  $options = array()) : string

NetCommons用Htmlを付加して、FormHelper::input()の結果を出力する<br> ※Overwrite FormHelper::input()

FormHelper->input()の$optionsに対する初期値

共通

array(
    'error' => false,
    'required' => null,
    'label' => null,
);

type=number の場合

array(
    'error' => false,
    'required' => null,
    'label' => null,
    'min' => 0,
    'div' => array('class' => 'form-group')
);

エラー発生時、入力ボックスの色を変えるためのdivを付与

<div class="has-error">
</div>
